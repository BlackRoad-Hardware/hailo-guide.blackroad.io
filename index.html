Installation
M.2 slot on Pi 5. Install hailo-firmware and hailort packages. Verify with 'hailortcli scan'. Should show /dev/hailo0.
›
Model Conversion
Convert ONNX → HEF format using Hailo Model Zoo. Quantize to INT8. Optimize for Hailo-8 architecture. Compile.
›
YOLOv5/v8
Pre-converted YOLO models available. Real-time object detection at 26 TOPS. Camera input via libcamera.
›
Performance
26 TOPS = 26 trillion operations per second. YOLOv8n at 30fps. Classification models at 100+ fps.
›
Power
~5W under load. Total Pi + Hailo draws ~10W max. Runs on standard USB-C power supply. No external power needed.
›
Integration
Python API: import hailo_platform. Load model, infer, get results. Integrates with Ollama for hybrid AI pipelines.
